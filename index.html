GREENMART

O seu novo app de compras sustentáveis.

Buscamos tornar a moda cada vez mais sustentável, através de movimentos como eco fashion, o meio de produção que evita a agressão ao meio-ambiente.

Quem Somos

O projeto GreenMart é um aplicativo desenvolvido para promover o consumo sustentável na moda, enfrentando os problemas do fast fashion, que contribui para poluição e condições de trabalho precárias. No GreenMart, você pode:

Comprar

Optar por peças sustentáveis e ajudar o meio-ambiente.

DIY

Aprender a dar vida nova às suas peças usadas.

Doar

Contribuir com ONGs que ajudam a melhorar a vida de muitas pessoas.

Funcionalidades

Doações

Dúvidas Frequentes

Algumas respostas para as principais perguntas.

Preciso pagar para utilizar o GreenMart?

Não! Acessar o app é gratuito.

Como faço para ganhar cupons?

É possível ganhar cupons ao fazer compras, ou então usando suas Greencoins para conseguir mais cupons de desconto!

Preciso ter uma conta para usar o app?

Não é necessário uma conta para ter acesso ao app, mas para fazer compras, doar e ganhar Greencoins, será preciso se cadastrar.

Quais tipos de roupas posso doar?

Aceitamos roupas limpas e em bom estado, prontas para serem reutilizadas. Evite peças danificadas ou inutilizáveis.
